Ketentuan Layanan

Selamat datang di Dunia Chatsyok!

1. Penerimaan Syarat

Dengan mendaftar, mengakses, atau bermain Chatsyok ("Layanan"), Anda setuju untuk terikat oleh Syarat dan Ketentuan ini. Jika Anda tidak setuju, mohon untuk tidak menggunakan layanan ini.

2. Kebijakan Privasi & Data Pengguna

Kami menghargai privasi Anda. Demi menjalankan fitur permainan (Save Progress, Level, Chat History), kami mengumpulkan dan menyimpan data berikut melalui autentikasi pihak ketiga (Google Firebase):

  • Identitas Publik: Nama Tampilan (Display Name), Alamat Email, dan Foto Profil (Avatar) yang terhubung dengan akun Google Anda.
  • Data Permainan: Statistik permainan seperti XP, Level, Rank, dan Pencapaian (Badge).
  • Riwayat Percakapan: Log obrolan (chat history) antara Anda dan Karakter AI disimpan di server kami untuk fitur "Ingatan Karakter" dan kelanjutan cerita.

Kami tidak akan pernah menjual data pribadi Anda kepada pihak ketiga. Email Anda hanya digunakan untuk keperluan identifikasi akun unik (Login).

3. Penggunaan Karakter AI (Artificial Intelligence)

Karakter dalam Chatsyok ditenagai oleh Kecerdasan Buatan (AI). Harap dipahami bahwa:

  • Semua balasan karakter dihasilkan secara otomatis oleh komputer.
  • Fiksi Sementata: Karakter mungkin melakukan "halusinasi" (mengarang fakta). Jangan jadikan ucapan karakter sebagai saran medis, hukum, atau finansial yang nyata.
  • Karakter didesain untuk hiburan semata.

4. Etika Pengguna (User Conduct)

Demi kenyamanan bersama, Anda dilarang keras untuk:

  • Menggunakan bahasa kasar, ujaran kebencian (SARA), atau konten seksual eksplisit secara berlebihan.
  • Mencoba meretas, memanipulasi (cheat), atau mengeksploitasi bug sistem untuk menaikkan Level/XP secara tidak wajar.
  • Melakukan spamming pada server API kami.

Pelanggaran terhadap poin ini dapat mengakibatkan Banned Permanen pada akun Anda tanpa peringatan.

5. Barang Virtual & Mata Uang

Fitur seperti "XP", "Level", "Koin", atau "Lencana" adalah aset virtual dalam game yang tidak memiliki nilai uang di dunia nyata. Aset ini tidak dapat ditukar dengan uang tunai atau dipindahkan ke akun lain.

6. Penafian (Disclaimer)

Layanan ini disediakan "sebagaimana adanya" (as-is). Pengembang (Dev) berhak untuk mengubah, menunda, atau menghentikan layanan kapan saja untuk keperluan pemeliharaan atau update tanpa pemberitahuan sebelumnya.

7. Kontak Pengembang

Jika Anda memiliki pertanyaan mengenai data Anda atau ingin mengajukan penghapusan akun, silakan hubungi tim pengembang melalui GitHub atau kontak yang tersedia di aplikasi.
